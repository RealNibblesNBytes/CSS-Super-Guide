Making a button wiggle upon touch.

Code: We can achieve this through an animation.

div.extraWiggleResult:hover{
 animation: wiggle 0.25s 1 normal ease-out;
}

@keyframes wiggle{
 0% { transform: rotate(-5deg); }
 25% { transform: rotate(3deg); }
 50% { transform: rotate(-2deg); }
 75% { transform: rotate(1deg); }
 100% { transform: rotate(0deg); }
}

Result: We made a button that wiggles.
